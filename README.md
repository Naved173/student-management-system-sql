# student-management-system-sql
Student Management System

A PostgreSQL-based Student Management System designed to manage student records, course details, enrollments, attendance, and marks efficiently. The database uses relational tables with primary and foreign key constraints to maintain data integrity and establish relationships between entities. The project demonstrates SQL concepts such as table creation, CRUD operations, JOINs, constraints, and database normalization.

Features
Student Information Management
Course Management
Student Course Enrollment Tracking
Attendance Management
Marks/Result Management
Relational Database Design with ER Diagram
SQL Queries using JOINs and Constraints
